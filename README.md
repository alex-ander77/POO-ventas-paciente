# Sistema POO - Ventas y Pacientes

Este proyecto fue desarrollado como práctica de Programación Orientada a Objetos (POO).

## Descripción
El sistema permite:
- Registrar información de vendedores
- Calcular comisiones e ingresos
- Evaluar el desempeño de ventas
- Registrar pacientes
- Calcular costos de atención médica
- Clasificar niveles de prioridad

## Tecnologías usadas
- Java

## Autor
- Alexander Lluglla

## Objetivo
Aplicar conceptos de:
- Encapsulamiento
- Clases y objetos
- Métodos get y set
- Constructores
- Lógica condicional
